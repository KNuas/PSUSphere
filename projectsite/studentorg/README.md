# PSUSphere

### Project Description
**PSUSphere** is a Django-based university management system designed to organize data for colleges, academic programs, student organizations, and memberships. This project demonstrates a full-stack development workflow, including database modeling, administrative customization, and automated data seeding.

### Features 
* **Environment Configuration**: Established a dedicated virtual environment (psusenv) and integrated the project with GitHub for collaborative version control.
* **Relational Database Modeling**: Developed five core models (*College, Program, Organization, Student,* and *OrgMember*) with automated timestamps and relational integrity.
* **Admin Dashboard Refactoring**: 
    * **Task A (College)**: Added searchable names and filters for creation dates.
    * **Task B (Program)**: Enabled searching by program and college name with a college-based filter.
    * **Task C (Organization)**: Displayed detailed descriptions and implemented college-specific filtering.
